# Restaurant Management System

## 📌 Overview
A full-stack restaurant management system with simulated API logic to manage orders, menu items, and customers.

## 🚀 Features
- Simulated API endpoints for handling data
- CRUD operations for menu and orders
- Order and customer management
- Structured backend logic

## 🛠 Tech Stack
- Backend: ASP.NET Core
- Database: SQL Server
- Frontend: (React / HTML CSS JS)

## 🔗 API Simulation
The system simulates API behavior using backend logic without full external API deployment.

## 📂 Project Structure
- Controllers → handle requests
- Models → database entities
- Services → business logic
- Data → database context

## ⚙️ How to Run
1. Clone the repository  
2. Open in Visual Studio  
3. Configure database  
4. Run the project
